# Derivatives of Polynomial Functions

| $\boldsymbol{f(x)}$ | $\boldsymbol{\frac{d}{dx} f(x)}$ |
|--|--|
| $a$ | $$0$$ |
| $ax+b$ | $$a$$ |
| $ax^2+bx+c$ | $$2ax+b$$ |
| $ax^3+bx^2+cx+d$ | $$3ax^2+2bx+c$$ |
| | $$x(3ax+2b)+c$$ |
| $$\large\sum_{i=1}^n\ \ a_i x^{n-1}$$ | $$\large\sum_{i=1}^{n-1}\ \ (n-i)a_i x^{n-i-1}$$ |
