# Desmos: Line Equation by Points Angles

| | |
|--|--|
| | circle |
| 1 | $r = 4$ |
| | getting points by angles |
| 2 | $P(\theta) = (r \cos(\theta), r \sin(\theta))$ |
| 3 | $P_1 = P\left( \frac{\pi}{4} \right)$ |
| 4 | $P_2 = P(\pi)$ |
| | getting line by points |
| 5 | $L(x,A,B) = (x-A.x)\frac{A.y-B.y}{A.x-B.x}+A.y$ |
| 6 | $f(x) = L(x,P_1,P_2)$ |
