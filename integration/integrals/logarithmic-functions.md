# Integrals of Logarithmic Functions

| $\boldsymbol{f(x)}$ | $\boldsymbol{\int f(x)\ dx}$ |
|--|--|
| $\ln(x)$ | $$x(\ln(x)-1)$$ |
| $\ln(ax)$ | $$x(\ln(ax)-1)$$ |
| $\ln(ax+b)$ | $$\left(\frac{b}{a}+x\right) \ln(ax+b)-x$$ |
| $\ln(ax^2)$ | $$x(\ln(ax^2)-2)$$ |
| $\ln(ax^2+b)$ | $$x(\ln(ax^2+b)-2)+\frac{2\sqrt{b} \arctan\left(\frac{\sqrt{a}\ x}{\sqrt{b}}\right)}{\sqrt{a}}$$ |
