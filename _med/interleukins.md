# Interleukins

| Interleukin | Main Source / Type | Property (Pro/Anti) | Main Mechanism / Function |
|-------------|--------------------|----------------------|----------------------------|
| **IL-1α/β** | Macrophages, epithelial cells | Pro-inflammatory | Fever, acute inflammation, activates endothelium |
| **IL-2** | Activated T cells | Pro | T-cell proliferation and survival |
| **IL-3** | T cells | Mixed | Hematopoiesis, supports myeloid precursors |
| **IL-4** | Th2 cells, mast cells | Anti/Mixed | IgE class switching, Th2 polarization |
| **IL-5** | Th2 cells | Pro | Eosinophil growth/activation |
| **IL-6** | Macrophages, endothelial cells | Pro | Acute-phase protein synthesis, fever |
| **IL-7** | Stromal cells | Mixed | T/B cell development and survival |
| **IL-8 (CXCL8)** | Macrophages, epithelial cells | Pro | Neutrophil chemotaxis |
| **IL-9** | Th9 cells | Mixed | Mast cell activation, mucosal immunity |
| **IL-10** | Tregs, macrophages | Anti | Inhibits Th1/Th17, suppresses inflammation |
| **IL-11** | Stromal cells | Anti/Mixed | Hematopoiesis, anti-inflammatory effects |
| **IL-12** | Dendritic cells, macrophages | Pro | Th1 differentiation, boosts IFN-γ |
| **IL-13** | Th2 cells | Mixed | Allergy, mucus production |
| **IL-14** | B-cell lymphomas, activated T cells | Mixed/Pro | B-cell proliferation, inhibits T cells |
| **IL-15** | Dendritic cells, macrophages | Pro | NK cell survival, memory CD8+ maintenance |
| **IL-16** | CD8+ T cells | Pro | Chemoattractant for CD4+ cells |
| **IL-17A–F** | Th17 cells | Pro | Neutrophil recruitment, chronic inflammation |
| **IL-18** | Macrophages | Pro | Enhances IFN-γ, synergizes with IL-12 |
| **IL-19** | Monocytes | Anti | Regulatory, suppresses inflammation |
| **IL-20** | Keratinocytes | Mixed | Tissue repair, epithelial response |
| **IL-21** | T follicular helper cells | Mixed | B-cell maturation, antibody production |
| **IL-22** | Th17, innate lymphoid cells | Mixed | Mucosal immunity, epithelial barrier repair |
| **IL-23** | Dendritic cells | Pro | Th17 stabilization and expansion |
| **IL-24** | Th2 cells, epithelial cells | Mixed | Tissue repair, apoptosis |
| **IL-25 (IL-17E)** | Th2 cells, epithelial cells | Pro | Promotes Th2 responses, allergy |
| **IL-26** | Th17 cells | Pro | Antimicrobial, induces inflammation |
| **IL-27** | Macrophages, dendritic cells | Anti/Mixed | Regulates immune activation, suppresses Th17 |
| **IL-28A/B (IFN-λ2/3)** | Epithelial cells | Pro | Antiviral responses |
| **IL-29 (IFN-λ1)** | Epithelial cells | Pro | Antiviral, mucosal protection |
| **IL-30 (IL-27p28)** | Macrophages | Anti/Mixed | Regulatory subunit of IL-27 family |
| **IL-31** | Th2 cells | Pro | Induces pruritus (itching), dermatitis |
| **IL-32** | T cells, NK cells | Pro | Induces TNF, IL-1, IL-6 |
| **IL-33** | Damaged epithelial cells (alarmin) | Pro | Th2 activation, allergy |
| **IL-34** | Monocytes | Mixed | Macrophage differentiation |
| **IL-35** | Tregs | Anti | Strong immunosuppression, inhibits Th17/Th1 |
| **IL-36 (A/B/G)** | Epithelial cells | Pro | Skin inflammation, psoriasis |
| **IL-37** | Monocytes | Anti | Global suppression of innate immunity |
| **IL-38** | Epithelial cells, B cells | Anti | IL-1 family regulatory cytokine |
| **IL-39** | B cells (proposed) | Pro | B-cell mediated inflammation (controversial) |
| **IL-40** | B cells | Mixed | B-cell development, mucosal immunity |
| **IL-41 (METRNL)** | Muscle, adipose tissue | Anti/Mixed | Tissue repair, eosinophil activation |
