# Interleukins

| Interleukin | Main Source/Type | Property | Main Mechanism/Function |
|-------------|--------------------|----------|----------------------------|
| **IL-1α/β** | macrophages, epithelial cells | $\textcolor{red} {pro}$ | fever, acute inflammation, activates endothelium |
| **IL-2** | activated T cells | $\textcolor{red}{pro}$ | T-cell proliferation and survival |
| **IL-3** | T cells | $\textcolor{blue}{mixed}$ | hematopoiesis, supports myeloid precursors |
| **IL-4** | Th2 cells, mast cells | $\textcolor{green}{anti}$ / $\textcolor{blue}{mixed}$ | IgE class switching, Th2 polarization |
| **IL-5** | Th2 cells | $\textcolor{red}{pro}$ | eosinophil growth/activation |
| **IL-6** | macrophages, endothelial cells | $\textcolor{red}{pro}$ | acute-phase protein synthesis, fever |
| **IL-7** | stromal cells | $\textcolor{blue}{mixed}$ | T/B-cell development and survival |
| **IL-8 (CXCL8)** | macrophages, epithelial cells | $\textcolor{red}{pro}$ | neutrophil chemotaxis |
| **IL-9** | Th9 cells | $\textcolor{blue}{mixed}$ | mast cell activation, mucosal immunity |
| **IL-10** | tregs, macrophages | $\textcolor{green}{anti}$ | strong suppression of Th1/Th17 |
| **IL-11** | stromal cells | $\textcolor{green}{anti}$ / $\textcolor{blue}{mixed}$ | hematopoiesis, anti-inflammatory |
| **IL-12** | dendritic cells, macrophages | $\textcolor{red}{pro}$ | Th1 differentiation, ↑IFN-γ |
| **IL-13** | Th2 cells | $\textcolor{blue}{mixed}$ | allergy, mucus production |
| **IL-14** | B cell lymphomas, activated T cells | $\textcolor{red}{pro}$ / $\textcolor{blue}{mixed}$ | B-cell proliferation, inhibits T-cells |
| **IL-15** | dendritic cells, macrophages | $\textcolor{red}{pro}$ | NK cell survival, memory CD8+ maintenance |
| **IL-16** | CD8+ T cells | $\textcolor{red}{pro}$ | chemoattractant for CD4+ cells |
| **IL-17A–F** | Th17 cells | $\textcolor{red}{pro}$ | neutrophil recruitment, chronic inflammation |
| **IL-18** | macrophages | $\textcolor{red}{pro}$ | enhances IFN-γ, synergizes with IL-12 |
| **IL-19** | monocytes | $\textcolor{green}{anti}$ | regulatory anti-inflammatory effects |
| **IL-20** | keratinocytes | $\textcolor{blue}{mixed}$ | tissue repair, epithelial response |
| **IL-21** | T follicular helper cells | $\textcolor{blue}{mixed}$ | B-cell maturation, antibody production |
| **IL-22** | Th17, ILCs | $\textcolor{blue}{mixed}$ | epithelial barrier, mucosal immunity |
| **IL-23** | dendritic cells | $\textcolor{red}{pro}$ | Th17 stabilization |
| **IL-24** | Th2 cells, epithelial cells | $\textcolor{blue}{mixed}$ | tissue repair, apoptosis |
| **IL-25 (IL-17E)** | Th2 cells, epithelial cells | $\textcolor{red}{pro}$ | promotes Th2 responses, allergy |
| **IL-26** | Th17 cells | $\textcolor{red}{pro}$ | antimicrobial, inflammatory induction |
| **IL-27** | macrophages, dendritic cells | $\textcolor{green}{anti}$ / $\textcolor{blue}{mixed}$ | regulates immune activation, suppresses Th17 |
| **IL-28A/B (IFN-λ2/3)** | epithelial cells | $\textcolor{red}{pro}$ | antiviral responses |
| **IL-29 (IFN-λ1)** | epithelial cells | $\textcolor{red}{pro}$ | antiviral, mucosal defense |
| **IL-30** | macrophages | $\textcolor{green}{anti}$ / $\textcolor{blue}{mixed}$ | regulatory subunit (IL-27 family) |
| **IL-31** | Th2 cells | $\textcolor{red}{pro}$ | pruritus (itching), dermatitis |
| **IL-32** | T cells, NK cells | $\textcolor{red}{pro}$ | induces TNF, IL-1, IL-6 |
| **IL-33** | damaged epithelial cells | $\textcolor{red}{pro}$ | alarmin, Th2 activation |
| **IL-34** | monocytes | $\textcolor{blue}{mixed}$ | macrophage differentiation |
| **IL-35** | tregs | $\textcolor{green}{anti}$ | strong immunosuppression |
| **IL-36 (A/B/G)** | epithelial cells | $\textcolor{red}{pro}$ | skin inflammation, psoriasis |
| **IL-37** | monocytes | $\textcolor{green}{anti}$ | broad suppression of innate immunity |
| **IL-38** | epithelial cells, B cells | $\textcolor{green}{anti}$ | IL-1 family anti-inflammatory |
| **IL-39** | B cells (proposed) | $\textcolor{red}{pro}$ | B-cell–mediated inflammation (experimental) |
| **IL-40** | B cells | $\textcolor{blue}{mixed}$ | B-cell development, mucosal immunity |
| **IL-41 (METRNL)** | muscle, adipose tissue | $\textcolor{green}{anti}$ / $\textcolor{blue}{mixed}$ | tissue repair, eosinophils |

> $\text{\color{red} pro}$ - pro-inflammatory  
> $\text{\color{green} anti}$ - anti-inflammatory
